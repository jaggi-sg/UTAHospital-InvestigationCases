Web Application for investigation of hospital admission cases
----------------

The application has 3 functions:
- Healthcare decision making-
  - Calculation of **deviation ratio & absolute difference from a large healthcare dataset** based on the user input provided in the web application. This helps in the healthcare decision making of the patients.

- Feature selection-
  - Identification of important features using **feature selection process** towards the prediction of discharge destination from the dataset using **Weka API**.

- Clinical decision support system-
  - Based on the user input from the web application, **association rule mining** is conducted on the selected input from the dataset to calculate confidence & support which helps in clinical decision support systems.
